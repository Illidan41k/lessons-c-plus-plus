- The code must be compiled as a C++11 application.

- The code should be compiled as a 64-bit application.
  Otherwise you may run out of memory when the genetic algorithm will
  be started with a very large population size.

- To compile this code with Visual Studio make sure to disable the use
  of precompiled headers:
  Project -> Properties -> Configuration Properties ->
  C/C++ -> Precompiled Headers
